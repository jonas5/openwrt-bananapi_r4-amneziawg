AmneziaWG OpenWrt Packages repository

This repository contains only the OpenWrt package directories for AmneziaWG:
- package/amneziawg-tools
- package/luci-proto-amneziawg
- package/kmod-amneziawg

Usage (local):
- Edit compile_openwrt.sh as needed, then run it. The script will clone OpenWrt if not provided and copy these package directories into the OpenWrt tree before building.
